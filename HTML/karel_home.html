Karel the robot

                          

Karel is a small simple robot. He lives in a very simple world too.

He's the hero of the game Karel the robot.

His world is bounded by walls and he's only able to walk on his waymarks top down and from left to right or reverse. These waymarks are characterized by little crosses.


You can control Karel by writing some code. Writing code is not as hard as you might think now. Just try it and you'll see Karel the robot is a lot of fun and learning to code is quite easy.

Learning to program isn't that hard as you might think now. It's quite easy to learn.
